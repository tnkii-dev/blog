Proyecto sexy-Knob

06 04 2004

Hice un módulo de input para html

Esta semana empecé a construír un reproductor de música para mi web, en algún momento quice implementar un control de volumen y pensé que un knob como los de FL-Studio quedaría bién, busque una librería pero ninguna de las opciones en la primera página de google era fácil de implementar o tenía buena documentación, así que mejor hice mi propio módulo.

El nombre no tiene nada de especial, los elementos custom necesitan un guion para funcionar según las reglas, escribí sexy como placeholder pero lo dejé así porque ya estaba muy avanzado el desarrollo cuando lo quise cambiar y porque honestamente no se me ocurrió nada mejor que ponerle XD

Publiqué unademo en mi web y el código fuente está disponible en elrepositorio de github.
